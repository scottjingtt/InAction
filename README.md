# InAction (ECCV 2022)
Implementation of the ECCV work "InAction" for interpretable driving behavior prediction. 

## Data
---
- [BDD-OIA](https://twizwei.github.io/bddoia_project/)
- [PSI](http://situated-intent.net/pedestrian_dataset/)

## Dependencies
---
- Python 3.6
- Pytorch 1.1
